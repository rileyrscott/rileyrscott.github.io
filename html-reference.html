HTML

Description:

The main code that holds the inputs and outputs of a website.

Syntax:

Start Tag: <html>
Content: Whole page
End Tag: <html>

Example:

<html>
<body>
<p> I Love ECS! </p>
<body>
<html>

Display:

I Love ECS!

w3schools

Head

Description:

This element can include a title for the document, scripts, styles, meta information, and more, it formats all heads.

Syntax:

Start Tag: <head>
Content: Desired Text
End Tag: <head>

Example:

<head>
<title> My Website </title>
<head>

Display:

My Website

Description:

This element is used to describe the content and theme of a website.

Syntax:

Start Tag: <tile>
Content: Desired Text
End Tag: <title>

Example:

<title> Pineapplenet </title>

Display:

Pineapplenet

Body

Description:

This element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, and more.

Syntax:

Start Tag: <body>
Content: Whole Page
End Tag: <bodyd>

Example:

<body>

<body>

Headings

Description:

Headings is used to make bigger/bolder titles.

Syntax:

Start Tag: <h1 or h2>
Content: Desired Text
End Tag: </h1 or /h2>

Example:

<h1> Define </h1>

Display:

Define

w3schools

Paragraph

Description:

This element represents a paragraph of a text, this is a block level element.

Syntax:

Start Tag: <p>
Content: Desired Text
End Tag: </p>

Example:

<p> This website was assigned and done by Riley Scott. It took some crucial time out of his precious and desired weekend. </p>

Display:

This website was assigned and done by Riley Scott. It took some crucial time out of his precious and desired weekend.

w3schools

Breaks

Description:

This element and tag will insert a line break.

Syntax:

Start Tag: <br>
Content: Desired Text
End Tag: <br>

Example:

<p> This <br> is <br> spaced </p>

Display:

This
is
spaced

Lines

Description:

This element and tag will insert a line in a desired location.

Syntax:

Start Tag: <hr>
Content: Line
End Tag: <hr>

Example:

<hr> <br> <hr>

Comments

Description:

This element is used to add a comment in html.

Syntax:

Start Tag: <!--
Content: Desired Text
End Tag: -->

Example:

<!-- This is secret and can't be viewed in the browser -->

Display:

This is secret and can't be viewed in the browser

w3schools

Links

Description:

This element is used to provide a clickable link to a desired location or other website.

Syntax:

Start Tag: <a href
Content: Desired Link
End Tag: </a>

Example:

<a href="http://bit.ly/2y2hrGc">Sling.gif </a>

Display:

Sling.gif w3schools

Lists

Description:

This element is used to make lists (varying in style and size) within an html website.

End Tag: </ul>

Tables

Description:

This is used to make tables, the HTML table consists of the <table> element and one or more <tr>, <th>, and <td> elements, it can also vary in appearance.

Syntax:

Start Tag: <table>
Content: Desired Text with <tr>, <th>, <td>
End Tag: <table>

Example:

<table>
<tr>
<th>Monday</th>
<th>Tuesday</th>
</tr>
<tr>
<td>3pm</td>
<td>10 books</td>
<tr>
<td>4pm</td>
<td>11 books</td>
</tr>
<table>

Display:

Monday Tuesday
3pm 10 books
4pm 11 books
