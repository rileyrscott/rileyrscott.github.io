Heading Element

Description

Heading is used to make bigger, more bold titles

Syntax

Start tag: <h1>
Content: Text
End tag: </h1>

Example

<h1> I love ECS!</h1>

Display

I love ECS!

Lists

Description

Used to make different lists

Syntax

Start tags: <ul> (bullets), <ol> (numbered items)
Content: <li>(item)</li>
End tags: </ul>, </ol>

Example

<ul>
<li>Apple</li>
<li>Carrot</li>
<li>other food</li>
</ul>

Display

  • Apple
  • Carrot
  • Other Food

W3SCHOOLS Link

Tables

Description

Used to make tables within a website

Syntax

Start tag: <table>(defining the table)
Content: <tr>(inside have the header and data)</tr>(table row), <td>(data)</td> (table data), <th>(header)<th>(table header)
End tag: </ul>, </ol>

Example

<table>
<tr>
<th>Firstname</th>
<th>Lastname</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td>
</tr>

Display

FirstnameLastname
JillSmith
